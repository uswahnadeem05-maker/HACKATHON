## 💡 How It Works (Tech)

### Data Flow
```
Job Market Data (Sample)
         ↓
Skill Extraction (NLP-ready)
         ↓
Gemini API (Personalization)
         ↓
Streamlit UI (Interactive)
         ↓
User Gets Personalized Path
```

### Key Feature: AI Recommendations
When user clicks "Get My Learning Path":
1. Collect: Current skills + Target role
2. Send to Gemini: "Analyze Pakistan job market. User has X skills, wants Y role. Recommend 3 skills with resources."
3. Gemini returns: Personalized recommendations
4. Display: Nice formatted text
